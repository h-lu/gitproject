# 简答题 SA1：偏差-方差权衡

## 题目

请解释机器学习中的**偏差-方差权衡（Bias-Variance Tradeoff）**，并回答以下问题：

1. **定义**：什么是偏差（Bias）和方差（Variance）？它们分别如何影响模型的预测性能？

2. **表现**：高偏差和高方差在模型训练和测试过程中分别有哪些典型表现？

3. **实践**：在实际应用中，如何通过以下方法来平衡偏差和方差？
   - 正则化（Regularization）
   - 数据量（Data Size）
   - 模型复杂度（Model Complexity）

## 评分标准

- **概念准确**（40%）：对偏差和方差的理解是否正确
- **要点覆盖**（35%）：是否涵盖了定义、表现和实践方法
- **表达清晰**（25%）：逻辑是否清晰，术语是否恰当
