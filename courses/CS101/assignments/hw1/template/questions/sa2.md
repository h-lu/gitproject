# 简答题 SA2：L1 vs L2 正则化

## 题目

请比较**L1 正则化（Lasso）**和**L2 正则化（Ridge）**，并回答以下问题：

1. **数学形式**：分别写出 L1 和 L2 正则化的数学表达式，并解释正则化项的作用。

2. **几何解释**：从几何角度解释为什么 L1 正则化倾向于产生稀疏解（sparse solution），而 L2 正则化倾向于产生较小的权重。

3. **特征选择**：L1 和 L2 正则化在特征选择方面有什么不同？在什么场景下应该选择 L1 而不是 L2？

4. **权重影响**：两种正则化方法对模型权重的影响有何不同？

## 评分标准

- **概念准确**（40%）：对 L1/L2 正则化的理解是否正确
- **要点覆盖**（35%）：是否涵盖了数学形式、几何解释、特征选择和权重影响
- **表达清晰**（25%）：逻辑是否清晰，术语是否恰当
