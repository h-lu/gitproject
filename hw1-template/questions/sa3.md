# 简答题 SA3：Precision/Recall/F1 与适用场景

## 题目

请解释二分类问题中的**精确率（Precision）**、**召回率（Recall）**和**F1 分数（F1-Score）**，并回答以下问题：

1. **定义**：基于混淆矩阵（Confusion Matrix），分别给出 Precision、Recall 和 F1 的数学定义。

2. **关系**：说明 Precision 和 Recall 之间的关系，以及为什么需要 F1 分数来综合评估模型性能。

3. **适用场景**：在以下场景中，应该优先考虑 Precision 还是 Recall？为什么？
   - 医疗诊断（如癌症检测）
   - 垃圾邮件过滤
   - 推荐系统（推荐商品给用户）

4. **阈值调整**：如何通过调整分类阈值来平衡 Precision 和 Recall？

## 评分标准

- **概念准确**（40%）：对 Precision/Recall/F1 的理解是否正确
- **要点覆盖**（35%）：是否涵盖了定义、关系、适用场景和阈值调整
- **表达清晰**（25%）：逻辑是否清晰，术语是否恰当

- 可以结合具体例子说明不同场景下的权衡
