# 客观题答案

请在 `my_answers.json` 中填写你的答案。

## 📝 答案格式

### 完整示例

```json
{
  "MC1": "A",
  "MC2": "B",
  "MC3": "C",
  "MC4": "B",
  "MC5": "C",
  "TF1": true,
  "TF2": false,
  "TF3": true,
  "TF4": true,
  "TF5": false,
  "MS1": ["A", "B", "C"],
  "MS2": ["A", "C", "D"],
  "MS3": ["A", "B", "C"],
  "FB1": "1 / (1 + e^(-z))",
  "FB2": "梯度",
  "FB3": ["训练误差", "测试误差"],
  "FB4": ["稀疏", "平滑"],
  "FB5": ["(TP + FP)", "(TP + FN)"]
}
```

## 📋 题型说明

### 1. 单选题（Multiple Choice）

- **题号**: MC1, MC2, MC3, ...
- **格式**: 单个字母（A/B/C/D）
- **示例**: `"MC1": "A"`
- **注意**: 大小写不敏感

### 2. 判断题（True/False）

- **题号**: TF1, TF2, TF3, ...
- **格式**: 布尔值（true/false，不加引号）
- **示例**: `"TF1": true`
- **注意**: 必须是 true 或 false，不能是字符串

### 3. 多选题（Multiple Select）

- **题号**: MS1, MS2, MS3, ...
- **格式**: 选项数组（["A", "B", "C"]）
- **示例**: `"MS1": ["A", "B", "C"]`
- **注意**: 
  - 选项顺序无关（["A", "B"] 和 ["B", "A"] 相同）
  - 必须完全正确才得分
  - 选多、选少、选错都不得分

### 4. 填空题（Fill in the Blank）

- **题号**: FB1, FB2, FB3, ...
- **格式**: 
  - 单个空: 字符串 `"答案"`
  - 多个空: 数组 `["答案1", "答案2"]`
- **示例**: 
  - `"FB1": "梯度"`
  - `"FB3": ["训练误差", "测试误差"]`
- **注意**: 
  - 不区分大小写
  - 自动去除首尾空格
  - 多空题答案顺序必须正确

## ✅ 答题要求

### JSON 格式要求

1. **使用双引号**：键和字符串值都用双引号
2. **布尔值不加引号**：true/false 不要写成 "true"/"false"
3. **数组格式**：多选题和多空填空题用方括号 []
4. **逗号分隔**：每个题目之间用逗号分隔
5. **最后一题不加逗号**：JSON 最后一项后面不要有逗号

### 常见错误

❌ **错误示例**:
```json
{
  "MC1": A,           // 缺少引号
  "TF1": "true",      // 布尔值不应加引号
  "MS1": "A, B",      // 应该用数组
  "FB1": '答案',      // 应该用双引号
  "FB5": "",          // 多空题应该用数组
}                     // 最后一项后不应有逗号
```

✅ **正确示例**:
```json
{
  "MC1": "A",
  "TF1": true,
  "MS1": ["A", "B"],
  "FB1": "答案",
  "FB5": ["答案1", "答案2"]
}
```

## 🚀 提交流程

### 1. 填写答案

在 `my_answers.json` 中填写你的答案。

### 2. 验证格式

使用 JSON 验证器检查格式是否正确：
- 在线工具：https://jsonlint.com/
- 或使用命令：`python3 -c "import json; print(json.load(open('my_answers.json')))"`

### 3. 提交

```bash
git add objective_answers/my_answers.json
git commit -m "完成客观题"
git push
```

### 4. 查看结果

提交后，在 Pull Request 中查看评分结果评论。

## 📊 评分说明

### 评分规则

- **单选题/判断题**: 选对得分，选错不得分
- **多选题**: 必须完全正确才得分（选多、选少都不得分）
- **填空题**: 答案必须完全匹配（不区分大小写）

### 评分反馈

评分结果包含：
- 总分和各题型得分
- 每道题的对错情况
- 错误题目的正确答案

## 💡 答题技巧

### 多选题

1. 仔细阅读题目，注意"哪些"字样
2. 每个选项独立判断
3. 不确定的选项不要勾选（宁缺毋滥）

### 填空题

1. 注意题目要求的格式
2. 数学公式注意符号和括号
3. 多个空注意顺序
4. 可以适当简化（如 "1/(1+e^(-z))" 和 "1 / (1 + e^(-z))" 都可以）

## ⚠️ 注意事项

1. **不要修改题号**：题号必须与题目文件一致
2. **不要遗漏题目**：确保所有题目都填写了答案
3. **检查格式**：提交前用 JSON 验证器检查格式
4. **及时提交**：完成后及时提交，避免迟交扣分
5. **查看反馈**：提交后及时查看评分结果，有问题及时修正
